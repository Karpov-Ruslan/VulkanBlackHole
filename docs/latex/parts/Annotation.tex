\begin{abstract}

    В данной работе исследуются методы визуализации неевклидовых пространств в режиме реального времени, на примерах искривленного пространства-времени в гравитационном поле черной дыры с помощью марширования лучей, трассировки лучей, а также предрасчитанных методов. Разработка методов ведется с использованием графического API Vulkan\cite{vulkan}.

    \vfill

\end{abstract}
